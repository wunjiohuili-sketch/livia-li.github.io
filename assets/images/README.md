# Images

这个文件夹用于存放网站图片。

## 个人照片

请将个人照片命名为：

```text
profile.jpg
```

当前网站会从这个路径读取个人照片：

```text
assets/images/profile.jpg
```

## 作品图片

首页封面图请按以下名称放入：

```text
work-01-cover.jpg
work-02-cover.jpg
work-03-cover.jpg
work-04-cover.jpg
work-05-cover.jpg
work-06-cover.jpg
```

详情图请按项目编号放入，例如：

```text
work-01-01.jpg
work-01-02.jpg
work-01-03.jpg
work-01-04.jpg
work-01-05.jpg
```

如果作品图片不存在，网页会显示 `IMAGE PENDING` 占位框。
